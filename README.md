🚀 Getting Started
Prerequisites

Install Python 3.x

Install Git

Clone the Repository
git clone https://github.com/ramnoa/PLP-PYTHON
cd plp-python-classwork

▶️ Running the Code

Navigate to the desired folder and run the Python file:

python filename.py


If a file requires additional libraries, install dependencies first:

pip install -r requirements.txt
